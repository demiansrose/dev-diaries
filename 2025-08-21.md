## 📓 Key lessons

- There are 2 types of lists in HTML: ordered (numbers) + unordered (bullet points)
```
<ol>
  <li>...</li>
</ol>

<ul>
  <li>...</li>
</ul>
```
- Use *a* + *href* elements to anchor links
```
<a href="LINK">DISPLAYED TEXT</a>
```
- Use *img* element to embed images
```
<img src="ADDRESS">
```
